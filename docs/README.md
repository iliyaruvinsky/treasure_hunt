# Treasure Hunt Analyzer Documentation

This directory contains documentation and context materials for the Treasure Hunt Analyzer (THA) system.

## Directory Structure

### `th-context/`
Core Treasure Hunt documentation and context files:
- `Treasure_Hunt_Findings_Report.pdf` - Sample findings report
- `Treasure_Hunt_Template.docx` - Report template
- `Treasure_Hunt_Campaign_Blurb.docx` - Marketing materials
- `Skywind_Treasure_Hunting_Methodology_and_PoC_Steps.pdf` - Methodology documentation
- `Integrated_SAP_Treasure_Hunt_Landing_Page.md` - Landing page content
- `readmore/` - Detailed readme files for each focus area

### `case-studies/`
Real-world case studies and examples:
- Alert Stories - Examples of alerts and their impact
- Case Studies - Client success stories (CBC, Maccabi, etc.)
- Cost Saving examples - Financial impact examples
- Brochures - Marketing and presentation materials

### `product-docs/`
Skywind platform product documentation:
- Skywind 4C documentation
- Skywind SoDA documentation
- Skywind JAM documentation
- Skywind AG documentation
- About Us materials

### `soda-templates/`
SoDA email templates for workflow notifications

These documents provide context for:
